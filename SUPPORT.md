# Support

## How to Get Help

### For Bugs or Feature Requests

Please open an issue using the appropriate template:

- [Bug Report](../../issues/new?template=00_bug_report.yml)
- [Feature Request](../../issues/new?template=02_enhancement.yml)
- [Documentation Issue](../../issues/new?template=01_documentation.yml)

### For Questions and Discussion

Use [GitHub Discussions](../../discussions) for:

- General questions about a project
- Ideas and proposals
- Show and tell

### For Security Issues

**Do not open a public issue for security vulnerabilities.**
Please review the [Security Policy](SECURITY.md) and follow the responsible
disclosure process described there.

## Response Times

This is an open-source project maintained in spare time. Response times are
best-effort. There are no SLAs. Contributions that fix bugs or add documentation
are always welcome.
